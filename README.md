Capital City Guessing Game

Project Description The Capital City Guessing Game is a Java-based quiz application designed to help students learn and revise the capital cities of different countries in an interactive way. Instead of memorizing from textbooks, users can test their knowledge through a simple question-and-answer game.\

The game includes three difficulty levels (Easy, Medium, and Hard), each offering different score values. Country and capital data are stored in text files and loaded using file handling concepts in Java.

Features

Three difficulty levels: Easy, Medium, and Hard
Randomly selected questions
Score calculation based on difficulty level
Uses file handling to load country–capital data
Case-insensitive and format-tolerant answer checking
Technologies Used

Java
File Handling (BufferedReader, FileReader)
Collections Framework (Map, List, ArrayList)
Scanner for user input How to Run the Program
Make sure Java JDK is installed on your system.

Save the following files in the same folder:

CapitalCityGame2.java
countries_level1.txt
countries_level2.txt
countries_level3.txt
Compile the program: javac CapitalCityGame2.java

Run the program: java CapitalCityGame2

Sample Data Format (Text Files) Each line in the text files should be written as: Country - Capital Example: India - New Delhi France - Paris Future Enhancements
